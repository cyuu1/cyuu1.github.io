Lists Usage
Unordered List An unordered list is shown, by default, with a bullet.This list used to present a list of items that not necessarily need to be shown in any specific order.
Ordered List An ordered list is shown, by default, with a number. This list used to present a list of items that need to be shown in a specific order.Example:steps to be followed to bake a chocolate cake.
Definition List A definition list comes in pairs-the term to be defined and the definition(description). This list is used for glossary, vocabulary, questions and answers, etc.
